read.edf.signals {MatchingPursuit}R Documentation

Reads a selected EDF or EDF+ file and returns all signals data

Description

The function reads a selected EDF or EDF+ file and returns all signals data as a matrix. Also resampling can be done (upsampling or downsampling).

Usage

read.edf.signals(file, resampling = TRUE, f.new = NULL, from = NULL, to = NULL)

Arguments

file

the full path to the EDF/EDF+ file to be read.

resampling

logical TRUE or FALSE. If TRUE the frequency of all signals will be upsampling or downsampling, depending on the actual sampling rate of subsequent channel.

f.new

a new frequency.

from

loading a signal from the given second.

to

loading a signal to the given second.

Details

If resampling=TRUE, the frequency of all signals will be upsampled or downsampled, depending on the actual sampling rate of the individual channels and the set value of the f.new parameter. The EDF standard assumes that each channel can be sampled at a different rate. Therefore, it may happen that some channels are upsampled and others are downsampled. The function does not provide the functionality to independently change the sampling rate for each channel.

Value

A matrix edf.mtx with all signals data is returned and a list edf as a result of executing edf::read.edf() function.

Examples

Run examples

file <- system.file("extdata", "EEG_data_10sec.edf", package = "MatchingPursuit")
sigs1  <- read.edf.signals(file, resampling = FALSE)
head(sigs1)
sigs2 <- read.edf.signals(file, resampling = TRUE, f.new = 128)
head(sigs2)
