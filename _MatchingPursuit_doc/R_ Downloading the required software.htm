empi.download {MatchingPursuit}R Documentation

Downloading the required software

Description

Downloads Enhanced Matching Pursuit Implementation program empi.

Usage

empi.download()

Details

empi is a fast implementation of Matching Pursuit algorithm. The program uses both CPU parallelization and GPU devices. See README.md for more details on this implementation. The program (sources and binaries for different OSs) is available for download at empi. Details are presented in a journal paper: Różański, P.T. (2024). empi: GPU-Accelerated Matching Pursuit with Continuous Dictionaries. ACM Transactions on Mathematical Software, vol.50, issue = 3, pp. 1-17, https://doi.org/10.1145/3674832.

Value

The function downloads the empi program in a version compatible with the operating system used (Windows, Linux, MacOS-x64, MacOS-arm64) and saves it in the current directory.
